# Rule: Project Overview — ToolSEO Planner

## Mục Tiêu Dự Án

**ToolSEO Planner** = công cụ **lập kế hoạch SEO/GEO** cho SEO Planner tại agency.

- **KHÔNG viết content** — chỉ planning, structuring, brief, checklist, export
- Triết lý cốt lõi: **Planning-first · Template-driven · Checklist-based**
- User: SEO Planner nội bộ (không phải client)
